HTML:

▼
  • Hypertext Markup Language (HTML) creates the structure of the webpage. It's composed of elements which work together to provide content and a basic framework for all your pages on the web
  • Elements can be used to divide the content on the page and to describe the content to browsers and other developers
  • You can think of HTML as the frame of our websites. Each HTML element performs a unique role (e.g., head and body)
  • A good resource for identifying different HTML elements is W3 School

CSS:

▼
  • CSS adds style to our webpage. Similar to how HTML is the frame/foundation of a house, CSS is the interior and exterior designer
  • You can include CSS in your HTML through in-line styling, id, class, and/or element types, which is in order of priority (e.g., in-line styling will appear over class styling for a particular element)
  • CSS code uses a selector (e.g., div) followed by a curly bracket and then a CSS property and a value (e.g., div {color: blue}, which is called a declaration)
  • A good resource for styling HTML elements with CSS is W3 School

Git:

▼
  • Git is a version control system to manage and keep track of each version of your project, which we can use through the command line
  • Through Git, we can connect to our GitHub repositories, make changes to them, and keep them in sync with our local environment
  • Git is useful for developers because it allows us to keep track of changes to the code base and revert back to prior versions in case of an issue
  • A good resource for different git commands is Git Cheat Sheet
  • A good resource for more information on Git and GitHub is GitHub

JavaScript:

▼
  • JavaScript is a high-level dynamic programming language that allows users to interact with websites. It takes static HTML/CSS and makes it interactive
  • Similar to the house analogy, you can think of JavaScript as language that makes your home a smart home
  • It allows you to include things such as conditional logic, arrays, loops, functions, etc.
  • A good resource for styling HTML elements with CSS is W3 School
